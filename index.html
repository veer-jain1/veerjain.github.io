Projects

Inverted Pendulum Control System

Designed and implemented a real-time inverted pendulum control system using an MPU6050 IMU and a PID-based feedback loop. A complementary filter fused accelerometer and gyroscope data to estimate tilt angle, while proportional control dynamically adjusted motor PWM output to maintain upright stability.

The system runs on a microcontroller with high-speed I2C communication, real-time sensor fusion, and motor direction control via an H-bridge. Gain tuning was performed live through serial input to study system response and stability.
